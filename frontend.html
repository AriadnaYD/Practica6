    • ¿Qué es Frontend?

    El término "frontend" se refiere a la parte de un sistema informático, aplicación web o software que interactúa directamente con el usuario. Es la interfaz de usuario visible y la experiencia que un usuario tiene al utilizar una aplicación o sitio web.
